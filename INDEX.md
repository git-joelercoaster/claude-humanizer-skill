## How to Use

### Simple Mode
```
Humanize this [genre]: [your text]
```
Auto-detects genre, applies rules, returns humanized version.

### Voice-Matched Mode
```
Humanize this. Here's my voice sample: [your writing]

[Text to humanize]
```
Calibrates to your writing style, then rewrites to match.

### Interactive Mode
```
Walk me through the changes on this [genre]:

[Your text]
```
Shows each AI pattern, explains reasoning, lets you decide on each change.

### Reverse Modes (After humanizing)
```
Make this more formal / casual / confident / hedged
```
